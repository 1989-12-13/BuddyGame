### Key Decisions
- Used sed for bulk color replacement (efficient for 155+ occurrences in GameScreen)
- Kept #fff for button text on colored backgrounds (red/amber buttons)
- Kept skin tone colors (#fadcbc, #c9a98b) in minigame body diagrams
- Used Python for reliable emoji replacement (sed had issues with multi-byte chars on Windows)
