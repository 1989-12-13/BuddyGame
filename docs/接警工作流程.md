接警工作流程

1、正常接警

1.1 报警响铃两声内接听电话，询问病人性别、年龄、姓名、地址、病情、接车地点等信息，并在 1分钟内完成派警。。

1.2 调度员应按标准接警流程进行接警，具体流程见图 1，其他特殊情况处理包括：

1.3报警人员无法提供具体地址时，调度人员应询问患者大概位置或方向并先行派车，再继续了解确切信息；

1.4若报警人员已明确告知地址，但电话挂断无法与其核实其他信息，且无法确定来电是否为谎报电话时，应先派车，再继续联系急救呼叫电话；

1.5若同一（或相近）地址连续急救呼叫，应认真核实患者地址、病情等信息，判断是否为同一事件，避免重派、漏派。

1.6若病家指定某医院，但医院距离病家较远，调度员应尽到告知义务；

1.7若排队电话未接起，应立即回拨核实是否需要急救。

2、派车

2.1调度员接警后，详细询问患者相关信息后，应按照就近、就急的调度原则进行派车， 应在 1分钟内发出派车指令，急救站应在收到命令单后3分钟内完成出车。

2.2派车后不应更改派车单，若需添加信息或与原派车单有不同之处，可在备注中注明并及时通知出车人员。

3、催车

调度人员接到催车电话时，应首先核实患者信息，并告知报警人员救护车正在驶向现场途中，注意接车并保持电话通畅。

4、转院

4.1急救站拨打，要询问地点，呼救类型，病情，报警电话，出车车辆。

4.2非住院病人报警，询问其所转医院，按正常接警流程问警、派车。

4.3住院病人报警，告知报警人转院需自行与所转医院联系，沟通床位、患者病情等相关信息，并告知报警人所转医院电话。

5、会车

调度人员接到会车电话时，应核实患者信息，询问会车信息（会车路线，车型、颜色、车牌号），及时通知救护车驾驶员，并告知报警人打开双闪；如报警人未与患者在一起，应询问随车电话，并告知救护车驾驶员。
